Content folder naming and format

1) Layer 2 story file
- File: layer2-story.txt
- Encoding: UTF-8
- Format:
TITLE: your title
META: your meta line
LINES:
line 1
line 2
*active line (prefix with *)
line 4

2) Card content file
- File: cards-content.txt
- Encoding: UTF-8
- Format:
[Hero Name]
TRAUMA: trauma-side text
QUOTE: hero-side quote
BASIC: detail basic info
LOW: detail low point
HIGHLIGHT: detail highlight

Repeat one block per hero.

3) Credits file
- File: credits.txt
- Encoding: UTF-8
- Format:
FOOTER_LINE_1: line one
FOOTER_LINE_2: line two

Word documents (.doc/.docx) are not read directly by the page.
Please export to .txt first.
